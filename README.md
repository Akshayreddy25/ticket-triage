# Ticket Triage — AI Customer Support Platform

A full-stack AI platform that automatically classifies, routes, and resolves customer support tickets using NLP and RAG.

## Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Python, FastAPI
- **Database:** PostgreSQL, pgvector
- **AI/ML:** scikit-learn, DistilBERT, OpenAI/Claude API
- **Infrastructure:** Docker, GitHub Actions

## Features
- Automatic ticket classification and urgency detection
- Confidence-aware routing with human escalation
- Similar ticket retrieval via vector search
- LLM-generated response drafts with citations
- Agent feedback loop and evaluation dashboard
- Role-based access control

## Getting Started
\`\`\`bash
git clone https://github.com/Akshayreddy25/ticket-triage
cd ticket-triage
docker-compose up --build
\`\`\`

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
